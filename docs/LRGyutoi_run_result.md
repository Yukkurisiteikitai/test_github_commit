* 公開鍵暗号方式  

【ざっくりイメージ】   
ペットボトルで例えると、ペットボトルの締め方と、ペットボトルの開け方が独特なペットボトルを作っておいて、そのペットボトルの閉め方は教えるけどペットボトルの開け方は全く教えずに秘密にしておくことで、ペットボトルの中身（データ）を見れるのはペットボトル作った人しかいないからいいよねっていう話

【細かいやつ】  
「公開鍵」と「秘密鍵」というペアの鍵を使う暗号技術です。誰でも入手できる「公開鍵」でデータを暗号化し、受信者だけが持つ「秘密鍵」で復号します。この仕組みは、暗号化だけでなく、秘密鍵で署名し公開鍵で検証する電子署名にも使われます。
